# Regression Scenarios

Usar estos escenarios cuando se cambie lógica de simulación.

## Escenario 1 — Impacto básico

1. Sala 1.
2. Lanzar fuerte contra dummy.
3. Debe recibir daño o morir.
4. Monedas aumentan si muere.

## Escenario 2 — Impacto débil

1. Arrastre corto.
2. Golpear dummy lentamente.
3. No debería hacer daño relevante.

## Escenario 3 — Foso

1. Caer en pit.
2. Perder vida.
3. Volver a última posición segura.
4. Velocidad cero.

## Escenario 4 — Barril

1. Tocar barril.
2. Explosión daña enemigos cercanos.
3. Barril no explota de nuevo.

## Escenario 5 — Limpieza de sala

1. Eliminar todos los enemigos.
2. Si no es última sala: `choosing-upgrade`.
3. Si es última sala: `victory`.

## Escenario 6 — Proyectiles

1. Cambiar a flecha.
2. Disparar a enemigo.
3. Proyectil se destruye o expira.
4. Enemigo recibe daño.
