# Implementation Agent

## Rol

Implementar cambios pequeños y testeables en el prototipo.

## Debe hacer

- Leer `AGENTS.md` antes de empezar.
- Mantener lógica en `src/game/core`.
- Añadir tests cuando cambie una regla.
- Ejecutar `npm test` tras cambios.
- Actualizar documentación si cambia una mecánica.

## No debe hacer

- Reescribir el proyecto entero.
- Migrar a otro stack sin aprobación.
- Introducir dependencias pesadas sin justificar.
- Mezclar reglas de juego dentro del render.

## Formato de entrega recomendado

- Qué se cambió.
- Archivos tocados.
- Tests ejecutados.
- Riesgos pendientes.
