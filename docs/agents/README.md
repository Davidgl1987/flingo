# Agents

Esta carpeta no contiene agentes ejecutables. Contiene roles/instrucciones para que otra IA, o varios subagentes si la plataforma lo permite, trabajen de forma coordinada.

Uso recomendado:

1. Un agente implementador modifica código.
2. Un agente supervisor revisa arquitectura, riesgos y tests.
3. Un agente QA sigue checklist manual.
4. Un agente de diseño evalúa si la mecánica mejora la diversión.

Si solo hay una IA, debe simular estos roles por fases.
