# QA Agent

## Rol

Validar que el juego funciona desde la perspectiva del usuario.

## Pruebas automáticas

Ejecutar:

```bash
npm test
npm run build
```

Si no se pueden ejecutar, anotar motivo exacto.

## Pruebas manuales mínimas

- Arrancar `npm run dev`.
- Jugar sala 1 con mouse.
- Cambiar a flecha y matar un enemigo.
- Cambiar a hechizo y matar un enemigo.
- Caer en un foso.
- Activar barril.
- Recoger moneda/poción.
- Elegir mejora.
- Completar o morir.

## Reporte

Para cada bug:

- pasos;
- resultado esperado;
- resultado actual;
- severidad;
- archivo sospechoso si se conoce.
