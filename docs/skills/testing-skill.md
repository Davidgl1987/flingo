# Skill: Testing

## Objetivo

Mantener confianza mientras se itera rápido.

## Tipos de tests

### Automáticos

- Core lógico.
- Daño.
- Progresión.
- Upgrades.
- Colisiones simples.

### Manuales

- Sensación de lanzamiento.
- Legibilidad visual.
- Control táctil.
- Diversión.

## Cuándo añadir test

Añadir test si se toca:

- daño;
- fosos;
- barriles;
- progresión de sala;
- upgrades;
- proyectiles;
- condiciones de victoria/derrota.
