# Skill: Gameplay Physics Tuning

## Objetivo

Ajustar movimiento, rebotes y fricción para que el lanzamiento sea satisfactorio.

## Variables principales

- `PLAYER_LAUNCH_POWER`
- `MAX_AIM_DISTANCE`
- `PLAYER_MAX_SPEED`
- `PLAYER_DAMPING`
- `ROOM_RESTITUTION`
- `STOP_SPEED`

## Procedimiento

1. Cambiar una sola constante.
2. Probar sala 1.
3. Probar sala con foso.
4. Preguntar: ¿he tenido control?
5. Preguntar: ¿ha habido demasiada espera?
6. Ejecutar tests.

## Recomendación

Para móvil suele funcionar mejor:

- menos velocidad máxima;
- más legibilidad;
- menos precisión necesaria;
- feedback de trayectoria más claro.
