# Skill: Gameplay Prototyping

## Objetivo

Avanzar rápido sin perder la capacidad de evaluar diversión.

## Método

1. Implementar la versión más simple de la mecánica.
2. Probar en una sala controlada.
3. Añadir feedback mínimo.
4. Testear reglas críticas.
5. Decidir si se queda, se ajusta o se elimina.

## Regla

Una mecánica nueva debe sentirse en menos de 30 segundos de juego.
